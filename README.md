# Apartment Rent Prices
Scraping data from apartments.com to train ML model that estimates rent prices for apartments in the Bay Area.

## Packages used for web scraping:
<ul>
  <li>selenium
  <li>re (regular expressions)
  <li>pandas
</ul>
